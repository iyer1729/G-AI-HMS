# G-AI-HMS: Generative AI-Enabled Human Motion Simulation

**G-AI-HMS** is a fork of [MotionGPT](https://github.com/OpenMotionLab/MotionGPT), adapted for ergonomic motion simulation, task-specific prompt alignment, and biomechanical evaluation. This repository enables high-fidelity motion generation from natural language using AI-generated prompts and evaluates alignment against MediaPipe-extracted ground truth using standard metrics (MPJPE, PA-MPJPE, DTW, Cosine Similarity).
